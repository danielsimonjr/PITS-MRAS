# H∞ adversary core (design spec)

**Goal:** Add the H∞ robust-control *core* — a Game Algebraic Riccati Equation
solver and an analytic worst-case-disturbance (adversary) head — closing the
"head" of gap G1 faithfully to the framework's by-construction conventions.
Release v0.4.5. The neural adversarial min-max *training loop* is explicitly out
of scope (deferred to v0.5.0).

**Date:** 2026-06-04
**Sub-project:** 6 (final) of the remaining v0.4.0 set.

## Decisions (from brainstorming)

- **Scope:** H∞ *core, analytic* — `solve_gare` + analytic `AdversaryHead` +
  robust gain, all deterministically verifiable. No neural min-max loop.
- **Disturbance matrix `D`:** defaults to `B` (matched / input-channel
  disturbance), configurable via an override.
- **`γ` handling:** fixed (caller-supplied); `solve_gare` returns the stabilizing
  PD solution or **raises** if `γ` is infeasible; `γ → ∞` recovers the CARE.
- **Solver:** Hamiltonian–Schur (direct, robust, standard for GAREs; handles the
  indefinite term; no convergence tuning).
- **Placement:** `solve_gare` beside `solve_care` in `utils/lyapunov.py`;
  `AdversaryHead` beside `CostateHead` in `models/critic.py`. No new files; H∞
  stays opt-in.
- **No** `hinf_warm_start` helper (YAGNI — not in the approved component set;
  callers warm-start a critic to the GARE `P` via the existing `set_P`).

## Math (the conventions to implement)

System `ẋ = Ax + Bu + Dw`, cost `J = ∫(xᵀQx + uᵀRu − γ²‖w‖²) dt`. The H∞ value
`V = xᵀPx` uses the GARE

    AᵀP + PA + Q − P·M·P = 0,    M = B R⁻¹ Bᵀ − γ⁻² D Dᵀ.

- Robust control gain `K = R⁻¹BᵀP` → `u* = −Kx`.
- Worst-case disturbance gain `L = γ⁻²DᵀP` → `w* = Lx = (1/2γ²) Dᵀ∇V`
  (since `∇V = 2Px`).
- Worst-case closed loop `A − BK + DL = A − M P` is Hurwitz for the stabilizing
  solution.
- As `γ → ∞`, `M → BR⁻¹Bᵀ` and the GARE → the CARE (`solve_care`).

## Components

### `solve_gare(A, B, Q, R, gamma, D=None) -> (P, K, L)` (`utils/lyapunov.py`)

NumPy in/out (matching `solve_care`). `D` defaults to `B`. Hamiltonian–Schur:

1. `M = B @ inv(R) @ B.T - (1/gamma**2) * D @ D.T`.
2. `H = [[A, -M], [-Q, -A.T]]` (2n×2n).
3. `T, Z, sdim = scipy.linalg.schur(H, output="real", sort="lhp")` — sort the
   eigenvalues with negative real part first.
4. If `sdim != n`: raise `ValueError` (γ infeasible — eigenvalues on/over the
   imaginary axis; no proper stabilizing subspace).
5. `U11, U21 = Z[:n, :n], Z[n:, :n]`; `P = U21 @ inv(U11)`; symmetrize
   `P = (P + P.T) / 2`.
6. Validate: `P` PD (`min eigval > 0`) and `A − M P` Hurwitz; else raise
   `ValueError` (γ infeasible). `K = inv(R) @ B.T @ P`; `L = (1/gamma**2) @ D.T @ P`.

Docstring notes the `γ → ∞ ⇒ CARE` limit and the feasibility/raise contract.

### `AdversaryHead(critic, D, gamma)` (`models/critic.py`)

`nn.Module` mirroring `CostateHead`. Registers `D` `[state_dim, dist_dim]` and
stores `gamma`. `forward(e: [batch, state_dim]) -> w*: [batch, dist_dim]`:

    grad_V = critic.gradient(e)          # ∇V̂ = 2 P e  (autograd, like CostateHead)
    w_star = (1 / (2 * gamma**2)) * (grad_V @ D)   # = γ⁻² Dᵀ P e

So Identity-style "by construction": the adversary IS the (scaled) critic
gradient projected through `Dᵀ`, never an independent network. Add to
`models/__init__` re-exports and the top-level `pits_mras.__init__` `__all__`.

## Testing (deterministic, TDD)

`tests/test_identity_costate.py` already houses the Riccati/identity tests; add
H∞ tests there (or a focused `test_hinf.py` — choose `test_hinf.py` to keep the
new surface self-contained):

1. **`test_solve_gare_satisfies_gare`** — random stable `A`, `B`, `Q=I`, `R=I`,
   feasible `γ`: GARE residual `‖AᵀP+PA+Q−PMP‖ ≈ 0`; `P` symmetric PD;
   `A − MP` Hurwitz.
2. **`test_solve_gare_large_gamma_recovers_care`** — large `γ` (e.g. 1e6):
   `solve_gare` `P ≈ solve_care` `P` (atol ~1e-3).
3. **`test_solve_gare_raises_on_infeasible_gamma`** — `γ` below `γ_min` (tiny
   `γ`): raises `ValueError`.
4. **`test_solve_gare_default_D_is_B`** — omitting `D` equals passing `D = B`.
5. **`test_adversary_head_equals_analytic_worst_case`** — critic `set_P(P_gare)`;
   `AdversaryHead(critic, D, γ)(e) ≈ γ⁻²·(D.T @ P @ eᵀ)ᵀ` and
   `≈ (1/2γ²)·(∇V̂ @ D)`; finite, correct shape `[batch, dist_dim]`.
6. `test_imports` / `test_smoke` / `__all__` updated for the new export.
